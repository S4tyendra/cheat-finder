# Document Similarity Analyzer
 - **Concepts:** Tokenization, Stemming/Lemmatization, Stop Word Removal, both CountVectorizer and TfidfVectorizer, and both Cosine and Jaccard Similarity
 - **Goal:** How different methods rate the similarity between the same documents.
 - **Practical Relevance**:How different vectorization and similarity techniques perform is fundamental in many NLP tasks like document matching, recommendation, and clustering.
